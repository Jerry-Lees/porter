# Porter has moved

Porter is now **Poreia** — part of the [Praxis Suite](https://github.com/PraxisSuite).

## New Home

> **[github.com/PraxisSuite/Poreia](https://github.com/PraxisSuite/Poreia)**

Poreia is the evolution of Porter — a dual-pane terminal file manager built for homelabs and sysadmins. Same tool, better name, new home.

## Why the Move?

Porter was built to solve a specific problem: moving configs between servers and building repeatable deployment packages for Labinator (now Poiesis). Along the way it became a full-featured SSH/SFTP file manager in its own right. Together with Poiesis and Theoria it forms the **Praxis suite**: an ecosystem of homelab management tools built around the idea that a homelab should not be a second job.

The name Poreia comes from Aristotle's concept of *poreia* — journey, passage, the act of carrying forward. It fits.
